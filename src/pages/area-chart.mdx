# Area Chart

The **AreaChart** component is a lightweight, responsive data visualization tool built using native SVG and **Framer Motion**. It's designed to be lean, performant, and deeply customizable through theme tokens.

## Examples

### Default Area Chart

<Preview>
<AreaChart
  data={[
    { label: 'Jan', value: 10 },
    { label: 'Feb', value: 25 },
    { label: 'Mar', value: 18 },
    { label: 'Apr', value: 35 },
    { label: 'May', value: 28 },
    { label: 'Jun', value: 45 },
  ]}
  series={[
    { key: 'value', label: 'Total Users', color: 'var(--color-blue-500)' }
  ]}
/>
</Preview>

```tsx
import { AreaChart } from 'dashkit-ui';

function Example() {
  const data = [{ label: 'Jan', value: 10 }, { label: 'Feb', value: 25 }];
  const series = [{ key: 'value', label: 'Total Users', color: 'var(--color-blue-500)' }];

  return <AreaChart data={data} series={series} />;
}
```

## API Reference

### AreaChart

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `data` | `Record<string, string \| number>[]` | - | Array of objects containing data keys. |
| `series` | `AreaChartSeries[]` | - | Configuration for each data series to render. |
| `height` | `number \| string` | `'100%'` | Height of the chart container. |
| `showGrid` | `boolean` | `true` | Display dashed horizontal lines. |
| `showLabels` | `boolean` | `true` | Display X-axis labels. |
| `showTooltip` | `boolean` | `true` | Enable interactive tooltips. |
| `animate` | `boolean` | `true` | Enable entrance animations. |
| `className` | `string` | - | Additional CSS for the container. |
