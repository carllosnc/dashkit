# Area Chart

The **AreaChart** component is a lightweight, responsive data visualization tool built using native SVG and **Framer Motion**. It's designed to be lean, performant, and deeply customizable through theme tokens.

## Examples

### Default Area Chart

<Preview>

    { label: 'Jan', value: 10 },
    { label: 'Feb', value: 25 },
    { label: 'Mar', value: 18 },
    { label: 'Apr', value: 35 },
    { label: 'May', value: 28 },

/>
</Preview>

```tsx
import { AreaChart } from 'dashkit-ui';

const data = [
  { label: 'Jan', value: 10 },
  { label: 'Feb', value: 25 },
  // ... more data
];

function Dashboard() {
  return (
    <AreaChart 
      data={data} 
      color="var(--color-blue-500)"
    />
  );
}
```

## API Reference

### AreaChart

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `data` | `AreaChartDataPoint[]` | - | Array of objects with `label` and `value`. |
| `height` | `number` | `200` | Height in pixels. |
| `color` | `string` | `currentColor` | Color for the line and gradient area. |
| `showGrid` | `boolean` | `true` | Display dashed horizontal lines. |
| `showLabels` | `boolean` | `true` | Display X-axis labels. |
| `animate` | `boolean` | `true` | Enable entrance animations. |
| `className` | `string` | - | Additional CSS for the container. |
