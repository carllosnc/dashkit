# Dropdown Menu

An interactive menu that displays a list of actions or options when triggered.

## Import

```tsx
import { 
  Dropdown, 
  DropdownTrigger, 
  DropdownContent, 
  DropdownItem, 
  DropdownLabel, 
  DropdownSeparator 
} from '@/components/Dropdown/Dropdown';
```

## Installation

```bash
bunx carllosnc/dashkit add dropdown
```

## Basic Usage

A simple dropdown with items, labels, and separators.

<Preview>
  <BasicMenuDemo />
</Preview>

```tsx
<Dropdown>
  <DropdownTrigger asChild>
    <Button variant="filled">
      Options <FiChevronDown />
    </Button>
  </DropdownTrigger>
  <DropdownContent>
    <DropdownItem leftIcon={<FiUser />}>Profile</DropdownItem>
    <DropdownItem leftIcon={<FiSettings />}>Settings</DropdownItem>
    <DropdownSeparator />
    <DropdownItem destructive>Logout</DropdownItem>
  </DropdownContent>
</Dropdown>
```

## Selectable Items

Dropdown items can show a selected state, typically used for choosing an option from a list.

<Preview>
  <SelectableMenuDemo />
</Preview>

```tsx
export function SelectableExample() {
  const [selected, setSelected] = useState('Personal');

  return (
    <Dropdown>
      <DropdownTrigger asChild>
        <Button variant="outlined">{selected}</Button>
      </DropdownTrigger>
      <DropdownContent>
        <DropdownItem 
          selected={selected === 'Personal'} 
          onClick={() => setSelected('Personal')}
        >
          Personal
        </DropdownItem>
        <DropdownItem 
          selected={selected === 'Work'} 
          onClick={() => setSelected('Work')}
        >
          Work
        </DropdownItem>
      </DropdownContent>
    </Dropdown>
  );
}
```

## Custom Triggers

The `asChild` prop allows any element to act as the dropdown trigger, from icon buttons to text links.

<Preview>
  <CustomTriggerDemo />
</Preview>

```tsx
{/* Icon Button */}
<Dropdown align="right">
  <DropdownTrigger asChild>
    <button className="...">
      <FiSettings size={20} />
    </button>
  </DropdownTrigger>
  <DropdownContent>
    <DropdownLabel>Settings</DropdownLabel>
    <DropdownItem>Account</DropdownItem>
    <DropdownItem>Security</DropdownItem>
  </DropdownContent>
</Dropdown>

{/* Text Link */}
<Dropdown>
  <DropdownTrigger asChild>
    <button className="text-sky-500 font-semibold underline">
      Actions <FiChevronDown />
    </button>
  </DropdownTrigger>
  <DropdownContent>
    <DropdownItem>Download PDF</DropdownItem>
    <DropdownItem>Export CSV</DropdownItem>
  </DropdownContent>
</Dropdown>
```

## Features

- **Animated**: Uses Framer Motion for smooth scale and opacity transitions.
- **Accessible**: Closes automatically when clicking outside or selecting an item. Supports `Escape` key.
- **Customizable**: Supports custom triggers via `asChild`, alignment (`left` or `right`), and icons.
- **Styled**: Beautiful default styles for light and dark modes.

## Component API

### DropdownContent
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `align` | `'left' \| 'right'` | `'left'` | The alignment of the dropdown menu relative to the trigger. |
| `className` | `string` | - | Additional CSS classes for the content container. |

### DropdownItem
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `leftIcon` | `ReactNode` | - | Icon positioned to the left of the item text. |
| `rightIcon` | `ReactNode` | - | Icon positioned to the right of the item text. |
| `destructive` | `boolean` | `false` | If true, applies red "destructive" styling. |
| `selected` | `boolean` | `false` | If true, applies a selected state with a checkmark. |
| `disabled` | `boolean` | `false` | Disables interaction with the item. |
| `onClick` | `() => void` | - | Callback when the item is clicked. |
