# Tooltip

A small popup that displays information when a user hovers over an element.

## Import

```tsx
import { 
  Tooltip, 
  TooltipTrigger, 
  TooltipContent 
} from '@/components/Tooltip/Tooltip';
```

## Installation

```bash
bunx carllosnc/dashkit add tooltip
```

## Basic Usage

Tooltips are triggered by hovering over the trigger element.

<Preview>
  <TooltipDemo />
</Preview>

```tsx
<Tooltip>
  <TooltipTrigger asChild>
    <Button variant="outlined">
      <FiInfo /> Hover me
    </Button>
  </TooltipTrigger>
  <TooltipContent>
    Display helpful information
  </TooltipContent>
</Tooltip>
```

## Side Positioning

Tooltips can be positioned on the `top`, `bottom`, `left`, or `right` of the trigger.

<Preview>
  <PositioningDemo />
</Preview>

```tsx
<Tooltip>
  <TooltipTrigger asChild>
    <Button variant="outlined">Left Tooltip</Button>
  </TooltipTrigger>
  <TooltipContent side="left">
    I am on the left!
  </TooltipContent>
</Tooltip>
```

## Features

- **Animated**: Uses Framer Motion for smooth entry and exit transitions.
- **Accessible**: Closes automatically when the mouse leaves the trigger.
- **Smart Positioning**: Automatically calculates the best position relative to the trigger.
- **Portalled**: Renders at the end of the document body to avoid z-index and overflow issues.
- **Design System Consistent**: Uses theme-aware popover tokens and radius variables.
