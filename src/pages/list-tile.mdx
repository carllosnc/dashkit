# List Tile

A flexible row component for building lists, menus, and settings panels. Supports leading icons or avatars, a title and subtitle text block, and a trailing slot for actions or indicators.

## Import

```tsx
import { ListTile } from '@/components/dashkit/ListTile'
```

## Installation

```bash
bunx carllosnc/dashkit add listtile
```

## Dividers

Use the `divider` prop to render a bottom border directly on each tile. This lets you manage separators per-tile without a wrapper, which is useful when bleed-edge layouts remove the container.

<Preview>
  <div className="w-full">
    <ListTile title="Profile" leading={<FiUser size={18} />} trailing={<FiChevronRight size={16} />} interactive divider />
    <ListTile title="Notifications" leading={<FiBell size={18} />} trailing={<FiChevronRight size={16} />} interactive divider />
    <ListTile title="Privacy" leading={<FiLock size={18} />} trailing={<FiChevronRight size={16} />} interactive />
  </div>
</Preview>

```tsx
<ListTile title="Profile" divider interactive />
<ListTile title="Notifications" divider interactive />
<ListTile title="Privacy" interactive />
```

You can also use `divider` on all but the last item via array index:

```tsx
{items.map((item, i, arr) => (
  <ListTile
    key={item.label}
    title={item.label}
    divider={i < arr.length - 1}
  />
))}
```
