# Badge

Visual indicators used for counts, statuses, or high-priority labels. Dashkit provides two distinct badge components optimized for different use cases.

## Import

```tsx
import { Badge, FloatBadge } from '@/components/Badge/Badge'
```

## Installation

Add the badge to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add badge
```

## Components

- **`Badge`**: A standalone indicator. Use this for status labels or tags in lists.
- **`FloatBadge`**: A wrapper that places a badge over another element (e.g., notification counts on icons).

## Standalone Badge

Use the `Badge` component for simple labels or indicators that don't need to be attached to another element.

```tsx
import { Badge } from 'dashkit-ui';

export default () => (
  <div className="flex gap-4">
    <Badge content="Live" color="success" pulse />
    <Badge content="Experimental" color="info" />
    <Badge content="Critical" variant="outline" color="error" />
  </div>
);
```

## Floating Badge 

Use the `FloatBadge` component to wrap an element and position a badge relative to its corners.

```tsx
import { FloatBadge, Button } from 'dashkit-ui';
import { FiMail } from 'react-icons/fi';

export default () => (
  <FloatBadge content={5} color="error" position="top-right">
    <Button variant="outlined" leftIcon={<FiMail />}>
      Messages
    </Button>
  </FloatBadge>
);
```

## Features

- **Automatic Clipping**: Numbers larger than the `max` prop (default 99) are automatically formatted with a plus sign (e.g., `99+`).
- **Pulse Effects**: Easily add a status pulse animation to dots or small content badges via the `pulse` prop.
- **Rhythmic Corners**: All text-based badges now use a modern `rounded-sm` style, while dots remain `rounded-full`.
- **Flexible Variants**: Choose between `solid` and `outline` appearances.
