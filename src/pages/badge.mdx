# Badge

Visual indicators used for counts, statuses, or high-priority labels. Dashkit provides two distinct badge components optimized for different use cases.

## Import

```tsx
import { Badge, FloatBadge } from '@/components/Badge/Badge'
```

## Installation

Add the badge to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add badge
```

## Standalone Badge

Use the `Badge` component for simple labels or indicators.

    <Badge dot pulse color="success" />

```tsx
import { Badge } from 'dashkit-ui';

export default () => (
  <div className="flex gap-4">
    <Badge content="Live" color="success" pulse />
    <Badge content="Experimental" color="info" />
    <Badge content="Critical" color="danger" />
    <Badge dot pulse color="success" />
  </div>
);
```

## Floating Badge 

Use the `FloatBadge` component to wrap an element and position a badge relative to its corners.

<Preview>
  <div className="flex gap-8 items-center">
    <FloatBadge content={5} color="danger">
      <Button variant="outlined" leftIcon={<FiMail size={18} />}>
        Messages
      </Button>
    </FloatBadge>
    <FloatBadge content={12} color="success">
      <div className="p-3 bg-ds-100 dark:bg-white/5 border border-ds-200 dark:border-white/5 rounded-xl text-ds-600 dark:text-ds-400">
        <FiBell size={24} />
      </div>
    </FloatBadge>
    <FloatBadge dot pulse color="success" position="bottom-right">
      <div className="w-12 h-12 bg-ds-100 dark:bg-ds-800 border border-ds-200 dark:border-ds-700 rounded-full flex items-center justify-center overflow-hidden">
        <FiUser size={24} className="text-ds-400" />
      </div>
    </FloatBadge>
    <FloatBadge content={120} max={99} color="danger" position="top-right">
      <div className="p-3 bg-ds-100 dark:bg-white/5 border border-ds-200 dark:border-white/5 rounded-xl text-ds-600 dark:text-ds-400">
        <FiMessageSquare size={24} />
      </div>
    </FloatBadge>
  </div>
</Preview>

```tsx
import { FloatBadge, Button } from 'dashkit-ui';
import { FiMail } from 'react-icons/fi';

export default () => (
  <FloatBadge content={5} color="danger" position="top-right">
    <Button variant="outlined" leftIcon={<FiMail />}>
      Messages
    </Button>
  </FloatBadge>
);
```

## Features

- **Automatic Clipping**: Numbers larger than the `max` prop (default 99) are automatically formatted with a plus sign (e.g., `99+`).
- **Pulse Effects**: Easily add a status pulse animation to dots or small content badges via the `pulse` prop.
- **Rhythmic Corners**: All text-based badges now use a modern `rounded-sm` style, while dots remain `rounded-full`.
- **Flexible Variants**: Choose between `solid` and `outline` appearances.
