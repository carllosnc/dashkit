# DatePicker

A minimal, animated date selection component built with native React and Framer Motion. Ideal for dashboards and forms that require single date selection with a premium look.

## Import

```tsx
import { DatePicker } from 'dashkit'
```

## Installation

Add the Date Picker to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add datepicker
```

## Basic Usage

The `DatePicker` component provides a clean calendar interface for date selection.

<Preview>
  <div className="max-w-xs">
    <DatePicker label="Select Birthday" />
  </div>
</Preview>

```tsx
import { DatePicker } from 'dashkit';
import { useState } from 'react';

function Example() {
  const [date, setDate] = useState<Date>();

  return (
    <div className="max-w-xs">
      <DatePicker 
        label="Select Birthday" 
        value={date} 
        onChange={setDate} 
      />
    </div>
  );
}
```

## Controlled State

Pass a `value` and `onChange` handler to fully control the selection.

<Preview>
  <div className="max-w-xs">
    <DatePicker 
      label="Controlled Picker" 
      description="The selected date is tracked in parent state"
      value={new Date()}
    />
  </div>
</Preview>
