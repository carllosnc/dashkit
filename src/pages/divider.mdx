# Divider

A versatile component to visually separate sections of content with horizontal or vertical lines.

## Import

```tsx
import { Divider } from '@/components/Divider/Divider';
```

## Installation

```bash
bunx carllosnc/dashkit add divider
```

## Usage

```tsx
import { Divider } from '@/components/Divider/Divider';

export default function App() {
  return (
    <div className="flex flex-col gap-6 w-full">
      <Divider />
      <Divider>OR</Divider>
      <div className="flex items-center h-10 gap-4">
        <span>Option A</span>
        <Divider orientation="vertical" />
        <span>Option B</span>
      </div>
    </div>
  );
}
```

## Overview

Dividers help create visual hierarchy by grouping related content or providing clear breaks between different sections.
