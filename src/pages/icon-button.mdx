# Icon Button

A premium, specialized button for icon-only actions with customizable variant and rounding options.

## Import

```tsx
import { IconButton } from '@/components/IconButton/IconButton'
```

## Installation

Add the icon button component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add icon-button
```

## Basic Usage

Icon buttons are compact triggers that display only an icon. They are perfect for toolbars and navigation.

<Preview>
  <div className="flex items-center gap-6">
    <IconButton icon={<FiBell size={20} />} variant="filled" onClick={() => {}} />
    <IconButton icon={<FiEdit size={20} />} variant="outlined" onClick={() => {}} />
    <IconButton icon={<FiSettings size={20} />} variant="soft" onClick={() => {}} />
    <IconButton icon={<FiShare2 size={20} />} variant="ghost" onClick={() => {}} />
  </div>
</Preview>

```tsx
import { FiBell, FiSettings, FiShare2 } from 'react-icons/fi';

<IconButton icon={<FiBell />} variant="filled" />
<IconButton icon={<FiEdit />} variant="outlined" />
<IconButton icon={<FiSettings />} variant="soft" />
<IconButton icon={<FiShare2 />} variant="ghost" />
```

## Variants

Choose between four distinct visual styles: `filled`, `outlined`, `soft`, and `ghost`.

<Preview>
  <div className="flex items-center gap-6">
    <IconButton icon={<FiEdit size={18} />} variant="filled" />
    <IconButton icon={<FiEdit size={18} />} variant="outlined" />
    <IconButton icon={<FiEdit size={18} />} variant="soft" />
    <IconButton icon={<FiEdit size={18} />} variant="ghost" />
  </div>
</Preview>

```tsx
<IconButton icon={<FiEdit />} variant="filled" />
<IconButton icon={<FiEdit />} variant="outlined" />
<IconButton icon={<FiEdit />} variant="soft" />
<IconButton icon={<FiEdit />} variant="ghost" />
```

## Shapes

The `rounded` prop allows you to switch between a rounded-square (default) and a full circular shape.

<Preview>
  <div className="flex items-center gap-12">
    <div className="flex flex-col gap-3 items-center">
      <IconButton icon={<FiTrash size={18} />} rounded={false} onClick={() => {}} />
      <span className="text-[10px] font-bold text-ds-400 uppercase tracking-wider">Default (MD)</span>
    </div>
    <div className="flex flex-col gap-3 items-center">
      <IconButton icon={<FiTrash size={18} />} rounded={true} onClick={() => {}} />
      <span className="text-[10px] font-bold text-ds-400 uppercase tracking-wider">Circular</span>
    </div>
  </div>
</Preview>

```tsx
<IconButton icon={<FiTrash />} rounded={false} />
<IconButton icon={<FiTrash />} rounded={true} />
```

## States

Icon buttons can be disabled to prevent user interaction while remaining visually consistent.

<Preview>
  <div className="flex items-center gap-6">
    <IconButton icon={<FiSettings size={18} />} disabled />
    <IconButton icon={<FiTrash size={18} />} variant="filled" disabled />
  </div>
</Preview>

```tsx
<IconButton icon={<FiSettings />} disabled />
<IconButton icon={<FiTrash />} variant="filled" disabled />
```
