# Icon Button

<p className="text-xl text-base-500 dark:text-base-400 mt-2 mb-8">
  A premium, specialized button for icon-only actions with customizable scale, variant, and rounding.
</p>

## Overview

The `IconButton` component provides a consistent container for circular or rounded square icon triggers. It's often used in toolbars, navigation bars, and as utility shortcuts throughout application layouts.

## Installation

```bash
bunx carllosnc/dashkit add icon-button
```

## Usage

```tsx
import { IconButton } from '@/components/dashkit';
import { FiEdit, FiTrash, FiShare } from 'react-icons/fi';

export default function Actions() {
  return (
    <div className="flex gap-4">
      <IconButton icon={<FiEdit />} variant="soft" size="md" />
      <IconButton icon={<FiShare />} variant="ghost" size="sm" />
      <IconButton icon={<FiTrash />} variant="filled" size="lg" rounded />
    </div>
  );
}
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `icon` | `ReactNode` | `-` | The icon component to render inside the button. |
| `variant` | `'filled' \| 'outlined' \| 'soft' \| 'ghost'` | `'soft'` | Visual style variant of the button. |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'sm'` | Box size and icon scale of the component. |
| `rounded` | `boolean` | `false` | If true, applies `rounded-full`, otherwise `rounded-xl`. |
| `disabled` | `boolean` | `false` | Whether the button is interactive. |
