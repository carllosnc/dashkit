# Bar Chart

The **BarChart** component is a powerful, animated data visualization tool built for comparing discrete categories or showing distribution across groups. It supports multiple grouped series, interactive tooltips, and smooth entrance animations.

## Examples

### Single Series Bar Chart

<Preview>
<BarChart
  data={[
    { label: 'Jan', value: 400 },
    { label: 'Feb', value: 300 },
    { label: 'Mar', value: 200 },
    { label: 'Apr', value: 278 },
    { label: 'May', value: 189 },
    { label: 'Jun', value: 239 },
  ]}
  series={[
    { key: 'value', label: 'Revenue', color: 'var(--color-blue-500)' }
  ]}
/>
</Preview>

```tsx
import { BarChart } from 'dashkit-ui';

function Example() {
  const data = [
    { label: 'Jan', value: 400 },
    { label: 'Feb', value: 300 },
    //...
  ];
  const series = [{ key: 'value', label: 'Revenue', color: 'var(--color-blue-500)' }];

  return <BarChart data={data} series={series} />;
}
```

## API Reference

### BarChart

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `data` | `Record<string, string \| number>[]` | - | Array of objects containing data keys. |
| `series` | `BarChartSeries[]` | - | Configuration for each data series to render. |
| `height` | `number \| string` | `'100%'` | Height of the chart container. |
| `barGap` | `number` | `4` | Spacing between individual bars within a group. |
| `rounded` | `boolean` | `true` | Apply rounded corners to bars. |
| `showGrid` | `boolean` | `true` | Display dashed horizontal lines. |
| `showLabels` | `boolean` | `true` | Display X-axis labels. |
| `showTooltip` | `boolean` | `true` | Enable interactive tooltips. |
| `animate` | `boolean` | `true` | Enable entrance animations. |
| `className` | `string` | - | Additional CSS for the container. |
