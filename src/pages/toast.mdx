# Toast

A succinct message that provides feedback about an operation in a small, non-intrusive card.

## Installation

Add the toast component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add toast
```

## Setup

Wrap your application (usually in `main.tsx` or `App.tsx`) with the `ToastProvider`.

```tsx
import { ToastProvider } from '@/components/Toast/Toast'

function App() {
  return (
    <ToastProvider position="bottom-right">
      <YourApp />
    </ToastProvider>
  )
}
```

## Import

```tsx
import { toast, ToastProvider, useToast } from '@/components/Toast/Toast'
```

## Basic Usage

Trigger a toast from anywhere by calling the `toast` function. Toasts automatically stack as cards to save space.

<Preview>
  <ToastExamples />
</Preview>

```tsx
import { toast } from '@/components/dashkit'

const notify = () => {
  toast({
    title: "Project Saved",
    description: "Your changes are now live.",
    type: "success",
  })
}
```

## Positioning

Supports multiple screen positions for different layout requirements.

<Preview>
  <PositionExamples />
</Preview>

```tsx
toast({ title: "Alert", position: "top-center" })
toast({ title: "Feedback", position: "top-right" })
```

## Features

- **Stacked Visualization**: Toasts stack vertically, revealing themselves as the top ones are dismissed.
- **Portalled Layout**: Renders at the document root to avoid z-index layering issues.
- **Framer Motion**: Smooth, spring-based entry/exit and layout animations.
- **Auto-dismiss**: Configurable duration with a default of 5 seconds.
