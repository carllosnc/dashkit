# Switch

A high-performance, premium toggle switch component for binary choices, with support for labels and descriptions.

## Import

```tsx
import { Switch } from '@/components/Switch/Switch'
```

## CLI

Add the switch component to your project using the Dashkit CLI:

```bash
npx dashkit-ui add switch
```

## Basic Usage

The basic switch with a label.

<Preview>
  <Switch label="Enable Tracking" />
</Preview>

```tsx
<Switch label="Enable Tracking" />
```

```tsx
<Switch 
  label="Background Updates"

```

## Controlled State

Manage the switch state using React's `useState`.

<Preview>
  <SwitchDemo />
</Preview>

```tsx
import { useState } from 'react'

export function Settings() {
  const [enabled, setEnabled] = useState(false)

  return (
    <Switch 
      label="Push Notifications" 
      checked={enabled} 
      onChange={(e) => setEnabled(e.target.checked)} 
    />
  )
}
```

## Props

Extends all native `CheckboxHTMLAttributes<HTMLInputElement>` (except `type`), plus these specialties:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Title text rendered to the right. |
| `description` | `string` | — | Description text rendered below the label. |
| `className` | `string` | — | Additional CSS classes for custom styling. |
| `ref` | `Ref<HTMLInputElement>` | — | Forwarded ref to the underlying checkbox input. |
