# Introduction

Welcome to **Dashkit UI**, a leaner, simpler React component library designed for the modern web. Dashkit prioritizes **low dependency counts** and **direct control** over heavy abstractions.

## Philosophy

Dashkit is built for developers who want a professional design without the baggage. Our core pillars are:

1.  **Zero-Heavy Dependencies**: We avoid massive runtime libraries. No heavy component frameworks or complex state managers. Just React and Tailwind CSS v4.
2.  **Copy-Paste Ownership**: Similar to shadcn/ui, you don't install Dashkit as a package. You own the code. Use our CLI to add components directly to your source for absolute customizability.
3.  **Simple & Sharp Aesthetics**: Meticulously designed for professional tools. No fluff, no unnecessary animations, just premium components with native dark mode support.
4.  **Lean Performance**: By leveraging Tailwind CSS v4's CSS-first approach, we ensure minimal bundle overhead and lightning-fast load times.

## Key Features

-   🎨 **Sharp Design System**: High-contrast, professional-grade UI powered by Inter.
    -   🌓 **Native Dark Mode**: Seamless support for light and dark themes with true-black (neutral-950) backgrounds.
-   🚀 **Tailwind CSS v4**: Built with the latest, fastest version of Tailwind.
-   📦 **Minimal Footprint**: We prefer standard React hooks and native browser APIs over external libraries. Most components have zero external dependencies.
-   💻 **Dashkit CLI**: Instantly add components to your project with a single command.

## Installation

Add components directly to your project using our CLI:

```bash
bunx carllosnc/dashkit add [component-name]
```

Or check the specific component pages for manual installation instructions.

## Getting Support

Dashkit is an open-source project. You can find the source code, report issues, or contribute on [GitHub](https://github.com/carllosnc/dashkit).
