# Introduction

Welcome to **Dashkit UI**, a premium React component library designed for building modern, high-performance dashboards and developer tools.

## Philosophy

Dashkit is built on three core pillars:

1.  **Aesthetics First**: Every component is meticulously designed to look professional and premium out of the box, with built-in dark mode support.
2.  **Developer Experience**: Simple APIs, full TypeScript support, and seamless Tailwind CSS integration.
3.  **Performance**: Minimal runtime overhead. Styles are powered by Tailwind CSS v4, and documentation is powered by MDX.

## Key Features

-   🎨 **Modern Design System**: A curated palette and typography powered by Inter.
-   🌓 **Native Dark Mode**: First-class support for light and dark themes with persistent user preferences.
-   🚀 **Tailwind CSS v4**: Built with the latest CSS-first version of Tailwind.
-   📦 **Radix Primitives**: many components are built on top of Radix UI for accessibility (WIP).
-   💻 **CLI Support**: Add components directly to your project using the native Dashkit CLI.

## Installation

Components are designed to be copied into your project. Use our CLI to add them instantly:

```bash
bunx dashkit-ui add [component-name]
```

Or check the individual component pages for manual installation instructions.

## Getting Support

Dashkit is an open-source project. You can find the source code, report issues, or contribute on [GitHub](https://github.com/carllosnc/dashkit).
