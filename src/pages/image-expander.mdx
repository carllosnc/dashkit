# Image Expander

A premium component that enables cinematic image expansion with smooth animations, backdrop blur, and caption support.

## Import

```tsx
import { ImageExpander } from '@/components/ImageExpander/ImageExpander';
```

## Installation

```bash
bunx carllosnc/dashkit add image-expander
```

## Examples

Building a clean gallery with expanding visuals.

<Preview>
  <ImageExpanderDemo />
</Preview>

```tsx
<ImageExpander 
  caption="Cinematic Waterfall"
  full={<img src="/large.jpg" className="w-auto h-auto max-h-[85vh] rounded-lg" />}
>
  <img src="/thumb.jpg" className="w-full h-full object-cover rounded-lg" />
</ImageExpander>
```

## Features

- **Spring Dynamics**: Natural, bouncy transitions powered by Framer Motion.
- **Glassmorphism**: Elegant backdrop blur and translucent overlays.
- **Compositional**: Supports any ReactNode as the thumbnail or full-size content.
- **Responsive**: Adapts perfectly to mobile and desktop viewports.
- **Esc to Close**: Closes automatically when the Escape key is pressed.
