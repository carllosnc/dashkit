# Float Action Menu

A premium floating action button that reveals an interactive menu card sliding from right to left.

## Usage

```tsx
import { FloatActionMenu } from 'dashkit-ui'
import { FiCommand, FiPlus, FiUsers } from 'react-icons/fi'

export function MyComponent() {
  return (
    <FloatActionMenu label="Actions" icon={<FiCommand />}>
      <div className="space-y-4">
        <button className="flex items-center gap-2 w-full p-2 hover:bg-ds-50 rounded-lg">
          <FiPlus className="size-4" />
          <span>New Project</span>
        </button>
        <button className="flex items-center gap-2 w-full p-2 hover:bg-ds-50 rounded-lg">
          <FiUsers className="size-4" />
          <span>Invite Team</span>
        </button>
      </div>
    </FloatActionMenu>
  )
}
```

## Positions

The menu can be positioned at any corner of the parent container or screen.

```tsx
<FloatActionMenu position="bottom-right" label="Bottom Right" />
<FloatActionMenu position="bottom-left" label="Bottom Left" />
<FloatActionMenu position="top-right" label="Top Right" />
<FloatActionMenu position="top-left" label="Top Left" />
```
