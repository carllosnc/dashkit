# Textarea

A multiline text input component with support for labels, error states, and auto-growing height.

## Import

```tsx
import { Textarea } from '@/components/Textarea/Textarea'
```

## Installation

Add the textarea component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add textarea
```

## Basic Usage

The basic usage of the textarea with a label and placeholder.

<Preview>
  <Textarea label="Biography" placeholder="Tell us about yourself..." />
</Preview>

```tsx
<Textarea 
  label="Biography" 
  placeholder="Tell us about yourself..." 
/>
```

## Auto-Grow

Use the `autoGrow` prop to allow the textarea to automatically adjust its height based on the content. This provides a better user experience for long-form content.

<Preview>
  <div className="w-full max-w-sm">
    <Textarea 
      label="Message" 
      placeholder="Type something long to see it grow..." 
      autoGrow 
    />
  </div>
</Preview>

```tsx
<Textarea 
  label="Message" 
  placeholder="Type away..." 
  autoGrow 
/>
```

## States

The textarea supports error and disabled states, just like the input component.

<Preview>
  <div className="flex flex-col gap-6 w-full max-w-sm">
    <Textarea 
      label="Notes" 
      error="Something went wrong" 
      defaultValue="Invalid input content..." 
    />
    <Textarea 
      label="Locked Field" 
      disabled 
      value="This content is read-only." 
      helperText="You cannot edit this information" 
    />
  </div>
</Preview>

```tsx
<Textarea 
  label="Notes" 
  error="Something went wrong" 
  defaultValue="Invalid content..." 
/>

<Textarea 
  label="Locked Field" 
  disabled 
  value="Read-only text." 
  helperText="You cannot edit this" 
/>
```

## Props

The component extends all native `TextareaHTMLAttributes<HTMLTextAreaElement>` props, plus the following:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Textarea label rendered above the field. |
| `autoGrow` | `boolean` | `false` | Whether the textarea should automatically adjust its height. |
| `error` | `string` | — | Error message displayed below the field. |
| `helperText` | `string` | — | Helper message displayed below the field. |
| `className` | `string` | — | Additional CSS classes for the textarea element. |
| `ref` | `Ref<HTMLTextAreaElement>` | — | Forwarded ref to the underlying `<textarea>` element. |
