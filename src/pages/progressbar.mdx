# ProgressBar

Animated, fluid indicators for tracking task completion, loading states, or data metrics with high precision.

## Import

```tsx
import { ProgressBar } from '@/components/ProgressBar/ProgressBar'
```

## Installation

Add the progress bar into your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add progressbar
```

## Basic Usage

A simple progress bar with a controlled value. It uses spring-based animations for width changes by default.

<Preview>
  <div className="w-full max-w-md">
    <ProgressBar value={65} showLabel />
  </div>
</Preview>

```tsx
<ProgressBar value={65} showLabel />
```

## Label Positions

Labels can be placed either above the bar or to the side, which is ideal for compact layouts like table cells.

<Preview>
  <div className="w-full max-w-md flex flex-col gap-8">
    <ProgressBar label="System Resources" value={45} showLabel labelPosition="top" />
    <ProgressBar label="Build Progress" value={88} showLabel labelPosition="side" />
  </div>
</Preview>

```tsx
// Stacked layout (default)
<ProgressBar value={45} showLabel labelPosition="top" />

// Inline layout
<ProgressBar value={88} showLabel labelPosition="side" />
```
