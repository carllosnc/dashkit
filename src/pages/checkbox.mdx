# Checkbox

A custom-styled checkbox component for binary choices, with support for labels and descriptions.

## Import

```tsx
import { Checkbox } from '@/components/Checkbox/Checkbox'
```

## CLI

Add the checkbox component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add checkbox
```

## Basic Usage

The basic checkbox with a label.

<Preview>

</Preview>

```tsx
<Checkbox label="Remember me" />
```

## Controlled

You can use the `checked` and `onChange` props to control the checkbox state with React.

<Preview>
  <CheckboxDemo />
</Preview>

```tsx
import { useState } from 'react'

export function ControlledCheckbox() {
  const [checked, setChecked] = useState(false)

  return (
    <Checkbox 
      label="Controlled Checkbox" 
      description={`Currently: ${checked ? 'Checked' : 'Unchecked'}`}
      checked={checked}
      onChange={(e) => setChecked(e.target.checked)}
    />
  )
}
```

## Props

The component extends all native `InputHTMLAttributes<HTMLInputElement>` props (excluding `type`), plus the following:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Label rendered to the right of the checkbox. |
| `description` | `string` | — | Additional context rendered below the label. |
| `className` | `string` | — | Additional CSS classes for the checkbox container. |
| `ref` | `Ref<HTMLInputElement>` | — | Forwarded ref to the underlying `<input>` element. |
