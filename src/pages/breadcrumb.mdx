# Breadcrumb

A secondary navigation component that helps users understand their current location in the application hierarchy.

## Import

```tsx
import { Breadcrumb } from '@/components/Breadcrumb/Breadcrumb'
```

## Installation

Add the breadcrumb to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add breadcrumb
```

## Basic Usage

Pass an array of `items` to the component. The last item is automatically styled as the active location.

<Preview>
  <Breadcrumb
    items={[
      { label: 'Home', href: '/' },
      { label: 'Components', href: '/' },
      { label: 'Breadcrumb' },
    ]}
  />
</Preview>

```tsx
<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Components', href: '/docs' },
    { label: 'Breadcrumb' },
  ]}
/>
```

## With Icons

You can add icons to each item for better visual guidance. In this example we use `react-icons/fi`.

<Preview>
  <Breadcrumb
    items={[
      { label: 'Home', href: '/', icon: <FiHome size={14} /> },
      { label: 'Settings', href: '/', icon: <FiSettings size={14} /> },
      { label: 'Profile', icon: <FiUser size={14} /> },
    ]}
  />
</Preview>

```tsx
import { FiHome, FiSettings, FiUser } from 'react-icons/fi';

<Breadcrumb
  items={[
    { label: 'Home', href: '/', icon: <FiHome size={14} /> },
    { label: 'Settings', href: '/', icon: <FiSettings size={14} /> },
    { label: 'Profile', icon: <FiUser size={14} /> },
  ]}
/>
```

## Custom Separator

Override the default chevron separator with any React node.

<Preview>
  <Breadcrumb
    separator={<span className="text-neutral-300 dark:text-neutral-700 mx-1">/</span>}
    items={[
      { label: 'Projects', href: '#' },
      { label: 'Dashkit', href: '#' },
      { label: 'Releases' },
    ]}
  />
</Preview>

```tsx
<Breadcrumb
  separator={<span className="text-neutral-300 dark:text-neutral-700 mx-1">/</span>}
  items={[
    { label: 'Projects', href: '#' },
    { label: 'Dashkit', href: '#' },
    { label: 'Releases' },
  ]}
/>
```
