# Breadcrumb

A secondary navigation component that helps users understand their current location in the application hierarchy.

## Import

```tsx
import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbSeparator,
  BreadcrumbEllipsis
} from '@/components/Breadcrumb'
```

## Installation

Add the breadcrumb to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add breadcrumb
```

## Basic Usage

The `Breadcrumb` component uses a composition pattern. `BreadcrumbItem` can automatically render as a link if an `href` is provided.

<Preview>
  <Breadcrumb>
    <BreadcrumbList>
      <BreadcrumbItem href="/">Home</BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem href="/docs">Components</BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem active>Breadcrumb</BreadcrumbItem>
    </BreadcrumbList>
  </Breadcrumb>
</Preview>

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem href="/">Home</BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem href="/docs">Components</BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem active>Breadcrumb</BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## With Icons

You can add icons to your breadcrumb items by passing them as children alongside the text.

<Preview>
  <Breadcrumb>
    <BreadcrumbList>
      <BreadcrumbItem href="/">
        <FiHome size={14} /> <span> Home </span>
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem href="/settings">
        <FiSettings size={14} /> <span> Settings </span>
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem active>
        <FiUser size={14} /> <span> Profile </span>
      </BreadcrumbItem>
    </BreadcrumbList>
  </Breadcrumb>
</Preview>

```tsx
import { FiHome, FiSettings, FiUser } from 'react-icons/fi';

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem href="/">
      <FiHome size={14} /> Home
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem href="/settings">
      <FiSettings size={14} /> Settings
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem active>
      <FiUser size={14} /> Profile
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Custom Separator

Override the default separator with any React node.

<Preview>
  <Breadcrumb>
    <BreadcrumbList>
      <BreadcrumbItem href="/">Home</BreadcrumbItem>
      <BreadcrumbSeparator>/</BreadcrumbSeparator>
      <BreadcrumbItem active>Components</BreadcrumbItem>
    </BreadcrumbList>
  </Breadcrumb>
</Preview>

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem href="/">Home</BreadcrumbItem>
    <BreadcrumbSeparator>/</BreadcrumbSeparator>
    <BreadcrumbItem active>Components</BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Collapsed (Ellipsis)

Use `BreadcrumbEllipsis` to show a collapsed state for long paths.

<Preview>
  <Breadcrumb>
    <BreadcrumbList>
      <BreadcrumbItem href="/">Home</BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem>
        <BreadcrumbEllipsis />
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem active>Active Page</BreadcrumbItem>
    </BreadcrumbList>
  </Breadcrumb>
</Preview>

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem href="/">Home</BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbEllipsis />
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem active>Active Page</BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## Simplified Usage (Legacy)

You can still use the `items` prop for simpler cases.

```tsx
<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Components', href: '/docs' },
    { label: 'Breadcrumb' },
  ]}
/>
```
