# Breadcrumb

A secondary navigation component that helps users understand their current location in the application hierarchy.

## Import

```tsx
import { Breadcrumb } from '@/components/Breadcrumb/Breadcrumb'
```

## Installation

Add the breadcrumb to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add breadcrumb
```

## Basic Usage

Pass an array of `items` to the component. The last item is automatically styled as the active location.

```tsx
<Breadcrumb
  items={[
    { label: 'Home', href: '/' },
    { label: 'Components', href: '/docs' },
    { label: 'Breadcrumb' },
  ]}
/>
```

## With Icons

You can add icons to each item for better visual guidance.

```tsx
import { Home, User } from 'lucide-react';

<Breadcrumb
  items={[
    { label: 'Home', href: '/', icon: <Home size={14} /> },
    { label: 'Profile', icon: <User size={14} /> },
  ]}
/>
```

## Custom Separator

Override the default chevron separator with any React node.

```tsx
<Breadcrumb
  separator={<span className="text-base-300 mx-1">/</span>}
  items={[
    { label: 'Projects', href: '#' },
    { label: 'Dashkit' },
  ]}
/>
```

## API

### Breadcrumb Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `items` | `BreadcrumbItem[]` | — | Array of navigation items. |
| `separator` | `ReactNode` | `<ChevronRight />` | Custom separator element between items. |
| `className` | `string` | — | Additional CSS classes. |

### BreadcrumbItem Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `label` | `ReactNode` | — | The text or element to display for the item. |
| `href` | `string` | — | Optional URL for the breadcrumb link. |
| `icon` | `ReactNode` | — | Optional icon to display before the label. |
| `active` | `boolean` | `false` | Whether the item is the current page. |
