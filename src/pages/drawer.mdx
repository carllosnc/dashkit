# Drawer

A premium panel component that slides in from the edges of the screen, providing a temporary surface for navigation, settings, or details. It uses a compositional pattern to give you full control over the header, content, and footer sections.

## Import

```tsx
import {
  Drawer,
  DrawerHeader,
  DrawerContent,
  DrawerFooter
} from '@/components/Drawer/Drawer'
```

## Installation

Add the drawer component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add drawer
```

## Basic Usage

The Drawer is controlled via the `isOpen` prop and requires an `onClose` callback. You can compose your drawer using the provided sub-components.

<Preview>
  <DrawerDemo />
</Preview>

```tsx
import { useState } from 'react'
import {
  Drawer,
  DrawerHeader,
  DrawerContent,
  DrawerFooter,
  Button
} from '@/dashkit'

export default function MyPage() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>Open Drawer</Button>

      <Drawer
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        position="right"
      >
        <DrawerHeader>Header</DrawerHeader>
        <DrawerContent className="min-h-[200px]">Content</DrawerContent>
        <DrawerFooter>Footer</DrawerFooter>
      </Drawer>
    </>
  )
}
```

## Positions

The component supports all four screen edges, with smooth spring animations tailored for each.

- `right` (Default): Slides from the right.
- `left`: Slides from the left.
- `top`: Slides from the top.
- `bottom`: Slides from the bottom.

## Features

- **Spring Physics**: Natural, bouncy transitions using Framer Motion.
- **Backdrop Blur**: Softly blurs content behind the drawer for focus.
- **Gesture Support**: Swipe the drawer towards its edge to close it instantly.
- **Body Lock**: Automatically prevents background scrolling when open.
- **Compositional**: Use `DrawerHeader`, `DrawerContent`, and `DrawerFooter` for modular layouts.
- **Responsive**: Adapts its width or height based on the `size` prop or position.

### Sub-components

The sub-components `DrawerHeader`, `DrawerContent`, and `DrawerFooter` all accept `children` and an optional `className`.

- `DrawerHeader`: Standard padding and vertical layout for titles/descriptions.
- `DrawerContent`: Auto-scrolling container with custom scrollbars.
- `DrawerFooter`: Bottom-aligned flex container for action buttons.
