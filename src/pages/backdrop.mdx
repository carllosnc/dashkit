# Backdrop

A blurred or semi-transparent full-screen overlay, often used behind modals, drawers, or to dim the background for focus.

## Import

```tsx
import { Backdrop } from '@/components/Backdrop/Backdrop'
```

## Installation

Add the backdrop to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add backdrop
```

## Basic Usage

The backdrop component is usually controlled by a state.

export const BasicExample = () => {
  const [show, setShow] = useState(false);
  return (
    <>
      <Button onClick={() => setShow(true)}>Show Backdrop</Button>
      <Backdrop show={show} onClick={() => setShow(false)} />
    </>
  );
};

## With Children

You can pass children to the backdrop, which will be centered on top of the overlay.

export const ChildrenExample = () => {
  const [show, setShow] = useState(false);
  return (
    <>
      <Button onClick={() => setShow(true)}>Show Overlay with Content</Button>
      <Backdrop show={show} onClick={() => setShow(false)}>
        <div className="bg-card p-6 rounded-lg shadow-xl max-w-sm text-center">
          <h3 className="text-lg font-bold mb-2 text-foreground">Centered Content</h3>
          <p className="text-muted-foreground mb-4">This content is perfectly centered on top of the backdrop.</p>
          <Button onClick={() => setShow(false)}>Close</Button>
        </div>
      </Backdrop>
    </>
  );
};
