# Card

Dashkit's Card is designed as a **composed wrapper**, providing total layout control through granular sub-components.

## Installation

Add the Card component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add card
```

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter
} from '@/components/Card/Card';
```

## Basic Usage

The card can be used as a simple container or with its sub-components for more structure.

<div className='not-prose'>
  <Card>
    <CardHeader>
      <CardTitle>Project Overhaul</CardTitle>
      <CardDescription>A comprehensive update to the dashboard architecture and visual identity.</CardDescription>
    </CardHeader>
    <CardContent>
      <p className="text-sm text-ds-500">A standard composed card using the modular component internal structure for full layout control. This design pattern allows for deep nesting and flexible content slots.</p>
    </CardContent>
    <CardFooter>
      <Button variant="outlined">Dismiss</Button>
      <Button variant="filled">View Details</Button>
    </CardFooter>
  </Card>
</div>

```tsx
<Card>
  <CardHeader>
    <CardTitle>Project Overhaul</CardTitle>
    <CardDescription>
      A comprehensive update to the dashboard architecture.
    </CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card content goes here.</p>
  </CardContent>
  <CardFooter>
    <Button variant="outlined">Dismiss</Button>
    <Button variant="filled">View Details</Button>
  </CardFooter>
</Card>
```

## Composable Components

Cards are built using sub-components to provide maximum flexibility for different use cases. You can compose the header content as needed.

<div className="not-prose">
  <Card className="w-full">
    <CardHeader>
      <div className="flex items-center justify-between gap-[30px]">
        <div className="flex flex-col gap-1">
          <CardTitle>System Analytics</CardTitle>
          <CardDescription>Real-time monitoring of global infrastructure and system health.</CardDescription>
        </div>
        <Badge content="Live" color="info" pulse />
      </div>
    </CardHeader>
    <CardContent>
      <div className="grid grid-cols-1 sm:grid-cols-2 gap-4">
        <div className="p-4 rounded-xl bg-ds-50 dark:bg-white/5 border border-ds-100 dark:border-white/5">
          <span className="text-[10px] font-bold text-ds-400 uppercase tracking-widest">Uptime</span>
          <p className="text-xl font-bold font-mono text-foreground">99.99%</p>
        </div>
        <div className="p-4 rounded-xl bg-ds-50 dark:bg-white/5 border border-ds-100 dark:border-white/5">
          <span className="text-[10px] font-bold text-ds-400 uppercase tracking-widest">Latency</span>
          <p className="text-xl font-bold font-mono text-foreground">12ms</p>
        </div>
      </div>
    </CardContent>
    <CardFooter>
      <Button variant="outlined">View Global Status</Button>
    </CardFooter>
  </Card>
</div>

```tsx
<Card>
  <CardHeader>
    <div className="flex items-start justify-between w-full">
      <div className="flex flex-col gap-1">
        <CardTitle>System Analytics</CardTitle>
        <CardDescription>Real-time infrastructure performance</CardDescription>
      </div>
      <Badge content="Live" color="info" pulse />
    </div>
  </CardHeader>
  <CardContent>
    {/* Custom grid or data visualization */}
  </CardContent>
  <CardFooter>
     <Button variant="outlined">View Global Status</Button>
  </CardFooter>
</Card>
```
