# Installation

The Dashkit CLI is the fastest way to add components to your project. It handles copying files, managing internal dependencies, and installing required NPM packages automatically.

## Theme & Colors

Dashkit uses a semantic design system powered by Tailwind CSS v4. To ensure components render correctly, verify your `index.css` contains the following base-color variables:

```css

  /* Base colors */
  --color-base: var(--color-neutral-950);
  --color-base-50: var(--color-neutral-50);
  --color-base-100: var(--color-neutral-100);
  --color-base-200: var(--color-neutral-200);
  --color-base-300: var(--color-neutral-300);
  --color-base-400: var(--color-neutral-400);
  --color-base-500: var(--color-neutral-500);
  --color-base-600: var(--color-neutral-600);
  --color-base-700: var(--color-neutral-700);
  --color-base-800: var(--color-neutral-800);
  --color-base-900: var(--color-neutral-900);
  --color-base-950: var(--color-neutral-950);

  /* Background */
  --color-base-page: var(--color-white);
  --color-base-page-dark: var(--color-base-950);

  /* Success */
  --color-base-success: var(--color-green-500);
  --color-base-success-dark: var(--color-green-400);

  /* Error */
  --color-base-error: var(--color-red-500);
```

## Quick Start

Add a component using the following command:

```bash
bunx carllosnc/dashkit add [component-name]
```

## Commands

### `add`

Copies a component and its Dashkit dependencies to your project.

```bash
bunx carllosnc/dashkit add <component> [options]
```

#### Arguments

- `<component>`: The name of the component to add (e.g., `button`, `card`, `modal`).

#### Options

- `-o, --output <dir>`: Specifies the output directory. Defaults to `src/components/dashkit`.
- `--no-install`: Prevents the CLI from automatically installing missing NPM dependencies.

## Dependency Management

The CLI intelligently manages two types of dependencies:

1.  **Registry Dependencies**: If a component depends on another Dashkit component (e.g., `Card` depends on `Badge`), the CLI will automatically add those as well.
2.  **NPM Dependencies**: The CLI detects your package manager (`bun`, `pnpm`, `yarn`, or `npm`) and checks your `package.json`. If required packages (like `framer-motion` or `lucide-react`) are missing, it will install them automatically.

## Troubleshooting

If you encounter issues with automatic installation, you can use the `--no-install` flag and manually install the dependencies listed in the CLI output.
