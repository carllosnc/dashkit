# Avatar

A versatile component for presenting user profile pictures, with built-in fallback Support and smooth loading transitions.

## Installation

Add the Avatar to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add avatar
```

## Basic usage

By default, the Avatar component will show initials as a fallback if the image source is not provided or fails to load.

<Preview>
  <div className="flex items-center gap-4">
    <Avatar src="https://images.unsplash.com/photo-1535713875002-d1d0cf377fde?w=80&h=80&fit=crop" alt="John Doe" />
    <Avatar alt="Jane Smith" />
    <Avatar fallback="JD" className="bg-blue-500 text-white" />
  </div>
</Preview>

```tsx
import { Avatar } from 'dashkit-ui'

function Example() {
  return (
    <>
      <Avatar src="..." alt="John Doe" />
      <Avatar alt="Jane Smith" />
      <Avatar fallback="JD" className="bg-blue-500 text-white" />
    </>
  )
}
```

## Sizes

Available in five standard sizes: `xs`, `sm`, `md` (default), `lg`, and `xl`.

    <Avatar size="lg" alt="LG" />

</Preview>

## Avatar Group

Cluster multiple avatars together with automatic truncation using the `max` prop.

<Preview>
  <div className="flex flex-col gap-8">
    <AvatarGroup max={3}>
      <Avatar src="https://images.unsplash.com/photo-1535713875002-d1d0cf377fde?w=80&h=80&fit=crop" />
      <Avatar src="https://images.unsplash.com/photo-1599566150163-29194dcaad36?w=80&h=80&fit=crop" />
      <Avatar src="https://images.unsplash.com/photo-1527980965255-d3b416303d12?w=80&h=80&fit=crop" />
      <Avatar src="https://images.unsplash.com/photo-1544005313-94ddf0286df2?w=80&h=80&fit=crop" />
    </AvatarGroup>

    <AvatarGroup max={2} size="sm">
      <Avatar fallback="A" />
      <Avatar fallback="B" />
      <Avatar fallback="C" />
      <Avatar fallback="D" />
    </AvatarGroup>
  </div>
</Preview>

```tsx
<AvatarGroup max={3}>
  <Avatar src="..." />
  <Avatar src="..." />
  <Avatar src="..." />
  <Avatar src="..." />
</AvatarGroup>
```
