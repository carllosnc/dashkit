# Button Group

The **ButtonGroup** component is used to cluster multiple buttons together in a single, cohesive unit. This is often used for toolbar actions, toggle switches, or pagination.

## Examples

### Horizontal Group

The default layout for a button group.

<Preview>
  <div className="flex flex-col gap-4 w-full">
    <span className="text-xs font-bold uppercase tracking-widest text-ds-400">Standard Buttons</span>
    <ButtonGroup>
      <Button variant="outlined">Daily</Button>
      <Button variant="outlined">Weekly</Button>
      <Button variant="outlined">Monthly</Button>
    </ButtonGroup>
  </div>

  <div className="flex flex-col gap-4 w-full">
    <span className="text-xs font-bold uppercase tracking-widest text-ds-400">Icon Buttons</span>
    <ButtonGroup>
      <Button variant="outlined" leftIcon={<FiAlignLeft size={16} />} />
      <Button variant="outlined" leftIcon={<FiAlignCenter size={16} />} />
      <Button variant="outlined" leftIcon={<FiAlignRight size={16} />} />
    </ButtonGroup>
  </div>
</Preview>

```tsx
import { Button, ButtonGroup } from 'dashkit-ui';

function Example() {
  return (
    <ButtonGroup>
      <Button variant="outlined">Daily</Button>
      <Button variant="outlined">Weekly</Button>
      <Button variant="outlined">Monthly</Button>
    </ButtonGroup>
  );
}
```

### Vertical Group

Set the `vertical` prop to `true` to stack buttons vertically.

<Preview>
  <ButtonGroup vertical>
    <Button variant="outlined" leftIcon={<FiBold size={16} />}>Bold</Button>
    <Button variant="outlined" leftIcon={<FiItalic size={16} />}>Italic</Button>
    <Button variant="outlined" leftIcon={<FiUnderline size={16} />}>Underline</Button>
  </ButtonGroup>
</Preview>

```tsx
<ButtonGroup vertical>
  <Button variant="outlined">Bold</Button>
  <Button variant="outlined">Italic</Button>
  <Button variant="outlined">Underline</Button>
</ButtonGroup>
```

### Button Variants

ButtonGroup works with all button variants (`filled`, `outlined`, `soft`).

<Preview>
  <ButtonGroup>
    <Button variant="soft">Option A</Button>
    <Button variant="soft">Option B</Button>
    <Button variant="soft">Option C</Button>
  </ButtonGroup>
</Preview>
