# Button Group

The **ButtonGroup** component is used to cluster multiple buttons together in a single, cohesive unit. This is often used for toolbar actions, toggle switches, or navigation. It features a modern, border-aware outlined design with dedicated selection states.

## Examples

### Segmented Control

One of the most common uses for a button group is to switch between different views or filters. Use `flex-1` on the items to create equal-width segments.

<Preview>
  <ButtonGroup className="w-full">
    <ButtonGroupItem className="flex-1" selected>Activity</ButtonGroupItem>
    <ButtonGroupItem className="flex-1">Analytics</ButtonGroupItem>
    <ButtonGroupItem className="flex-1">History</ButtonGroupItem>
  </ButtonGroup>
</Preview>

```tsx
<ButtonGroup className="w-full">
  <ButtonGroupItem className="flex-1" selected>Activity</ButtonGroupItem>
  <ButtonGroupItem className="flex-1">Analytics</ButtonGroupItem>
  <ButtonGroupItem className="flex-1">History</ButtonGroupItem>
</ButtonGroup>
```

### View Switcher

Combine icons and text for a more descriptive interface. The selection state uses a subtle primary background to indicate the active choice.

      Grid

      List
    </ButtonGroupItem>

      <FiLayout size={16} />

</Preview>

```tsx
<ButtonGroup>

  <ButtonGroupItem className="gap-2">
```

### Formatting Toolbar

Button groups are essential for toolbars. The `-space-x-px` logic ensures borders never double up, maintaining a crisp appearance.

<Preview>
  <div className="flex gap-4">
    <ButtonGroup>
      <ButtonGroupItem selected><FiBold size={16} /></ButtonGroupItem>
      <ButtonGroupItem><FiItalic size={16} /></ButtonGroupItem>
      <ButtonGroupItem><FiUnderline size={16} /></ButtonGroupItem>
    </ButtonGroup>

    <ButtonGroup>
      <ButtonGroupItem selected><FiAlignLeft size={16} /></ButtonGroupItem>
      <ButtonGroupItem><FiAlignCenter size={16} /></ButtonGroupItem>
      <ButtonGroupItem><FiAlignRight size={16} /></ButtonGroupItem>
    </ButtonGroup>
  </div>
</Preview>

### Vertical Stack

Vertical groups are useful for sidebars or property menus.

<Preview>
  <ButtonGroup vertical>
    <ButtonGroupItem className="justify-start px-6 h-10">Top Action</ButtonGroupItem>
    <ButtonGroupItem className="justify-start px-6 h-10" selected>Active State</ButtonGroupItem>
    <ButtonGroupItem className="justify-start px-6 h-10">Bottom Action</ButtonGroupItem>
  </ButtonGroup>
</Preview>

```tsx
<ButtonGroup vertical>
  <ButtonGroupItem className="justify-start">Top Action</ButtonGroupItem>
  <ButtonGroupItem className="justify-start" selected>Active State</ButtonGroupItem>
  <ButtonGroupItem className="justify-start">Bottom Action</ButtonGroupItem>
</ButtonGroup>
```
