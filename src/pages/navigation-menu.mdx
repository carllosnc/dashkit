# Navigation Menu

A premium navigation menu component inspired by modern, high-end dashboards. It features a shared viewport that smoothly moves and resizes to accommodate different content sections.

## Import

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
  NavigationMenuViewport
} from '@/components/NavigationMenu/NavigationMenu'
```

## Installation

Add the navigation menu to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add navigation-menu
```

## Basic Usage

The Navigation Menu uses a value-based system to track the active item. Ensure the `value` on `NavigationMenuTrigger` matches the `value` on `NavigationMenuContent`.
