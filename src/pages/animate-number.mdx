# Animate Number

A specialized component for creating sleek numeric counters with independent digit animations. It uses a "top-to-bottom" sliding effect for each character, providing a premium and dynamic feel for dashboards and metrics.

## Import

```tsx
import { AnimateNumber } from '@/components/AnimateNumber/AnimateNumber'
```

## Installation

Add the animate-number to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add animate-number
```

## Basic Usage

The component will automatically animate whenever the `value` prop changes. Each digit that differs from the previous value will slide vertically to its new position.

<Preview>
  <AnimateNumberExample />
</Preview>

```tsx
import { useState } from 'react';
import { AnimateNumber, Button } from 'dashkit';

export default () => {
  const [val, setVal] = useState(1250);
  return (
    <div className="flex flex-col items-center gap-4">
      <div className="text-5xl font-bold">
        <AnimateNumber value={val} />
      </div>
      <Button onClick={() => setVal(v => v + 10)}>Increment</Button>
    </div>
  );
};
```

## Features

- **Character Splitting**: Each digit animates independently for a premium odometer feel.
- **Top-to-Bottom Movement**: New values slide in from above while old ones exit below.
- **Localization Support**: Automatically handles thousands separators and decimal markers via `Intl.NumberFormat`.
- **Customizable Duration**: Control the speed of each character's vertical transition.
