# Dock

A fluid, magnified navigation bar inspired by macOS, using Framer Motion for high-fidelity interactive physics.

## Import

```tsx
import { Dock, DockItem } from '@/components/Dock/Dock';
```

## Installation

```bash
bunx carllosnc/dashkit add dock
```

## Basic Usage

The dock is composed of a `Dock` container and multiple `DockItem` components.

<Preview>
  <DockDemo />
</Preview>

```tsx
<Dock>
  <DockItem icon={<FiHome />} label="Home" />
  <DockItem icon={<FiCommand />} label="Terminal" />
  <DockItem icon={<FiGrid />} label="Dashboard" />
  <DockItem icon={<FiGithub />} label="GitHub" href="https://github.com" />
</Dock>
```

## Theme Inversion

The Dock component implements an inverted contrast theme by default:
- **Light mode**: Uses dark backgrounds and light icons/text.
- **Dark mode**: Uses white backgrounds and dark icons/text.

This makes the dock stand out visually from the rest of the UI, behaving more like a floating control utility.

## Customization

You can adjust the `magnification` (how much it grows) and the `distance` (how far the magnet reach is).

```tsx
<Dock magnification={80} distance={200}>
  <DockItem icon={<FiHome />} />
</Dock>
```

## Features

- **Fluid Magnification**: Uses `framer-motion`'s `useSpring` and `useTransform` for physics-based scaling.
- **Smart Positioning**: Items adjust their width relative to the cursor position on the `X` axis.
- **Tooltips Included**: Built-in hover labels for quick identification of icon-only items.
- **Theme Logic**: Specialized colors that provide high contrast in both themes.
- **Composed API**: Clean JSX structure for easy item management.
