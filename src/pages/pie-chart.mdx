# Pie Chart

The **PieChart** component is an expressive tool for visualizing categorical distributions. It features interactive segments with tooltips, smooth hover scaling, and supports donut variations via the `innerRadius` prop.

## Examples

### Classic Pie Chart

<Preview>
<PieChart
  data={[
    { label: 'Chrome', value: 400, color: 'var(--color-blue-500)' },
    { label: 'Firefox', value: 300, color: 'var(--color-orange-500)' },
    { label: 'Safari', value: 200, color: 'var(--color-sky-400)' },
    { label: 'Edge', value: 100, color: 'var(--color-emerald-500)' },
  ]}
/>
</Preview>

```tsx
import { PieChart } from 'dashkit-ui';

function Example() {
  const data = [
    { label: 'Direct', value: 400, color: '#0ea5e9' },
    { label: 'Social', value: 300, color: '#f59e0b' },
    //...
  ];

  return <PieChart data={data} />;
}
```

### Donut Chart

Transform any pie chart into a donut chart by setting the `innerRadius` property (0 to 1).

<Preview>
<PieChart 
  innerRadius={0.6}
  data={[
    { label: 'Active', value: 45, color: '#10b981' },
    { label: 'Inactive', value: 52, color: '#64748b' },
    { label: 'Pending', value: 48, color: '#3b82f6' },
  ]}
/>
</Preview>

```tsx
<PieChart 
  innerRadius={0.6} 
  data={userData} 
/>
```

## API Reference

### PieChart

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `data` | `PieChartDataItem[]` | - | Array of objects with label, value, and color. |
| `height` | `number \| string` | `300` | Height of the chart container. |
| `innerRadius` | `number` | `0` | Radius of the center hole (0 for Pie, > 0 for Donut). |
| `showLabels` | `boolean` | `true` | Display the legend below the chart. |
| `showTooltip` | `boolean` | `true` | Enable interactive segment tooltips. |
| `animate` | `boolean` | `true` | Enable entrance animations. |
| `className` | `string` | - | Additional CSS for the container. |
