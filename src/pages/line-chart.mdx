# Line Chart

The **LineChart** component is a clean, minimal data visualization tool built using native SVG and **Framer Motion**. It's ideal for showing precise trends without the visual weight of an area fill.

## Examples

### Default Line Chart

<Card className='not-prose'>
<LineChart
  data={[
    { label: 'Jan', value: 10 },
    { label: 'Feb', value: 25 },
    { label: 'Mar', value: 18 },
    { label: 'Apr', value: 35 },
    { label: 'May', value: 28 },
    { label: 'Jun', value: 45 },
  ]}
  series={[
    { key: 'value', label: 'Sessions', color: 'var(--color-indigo-500)' }
  ]}
/>
</Card>

```tsx
import { LineChart } from 'dashkit';

function Example() {
  const data = [{ label: 'Jan', value: 10 }, { label: 'Feb', value: 25 }];
  const series = [{ key: 'value', label: 'Sessions', color: 'var(--color-indigo-500)' }];

  return <LineChart data={data} series={series} />;
}
```

### Growth Comparison (Multi-Line)

Analyzing multiple growth vectors simultaneously to identify correlations between revenue, costs, and percentage growth.

<Card className='not-prose'>
<LineChart
  data={[
    { label: 'Q1', revenue: 4500, expenses: 3200, profit: 1300, growth: 1200 },
    { label: 'Q2', revenue: 5200, expenses: 3800, profit: 1400, growth: 1500 },
    { label: 'Q3', revenue: 4800, expenses: 4500, profit: 300,  growth: 400 },
    { label: 'Q4', revenue: 6100, expenses: 4200, profit: 1900, growth: 1800 },
  ]}
  series={[
    { key: 'revenue',  label: 'Direct Revenue', color: '#3B82F6' },

    { key: 'growth',   label: 'Growth Trend',   color: '#8B5CF6' }
  ]}
/>
</Card>
