# Sidebar

A versatile and premium navigation sidebar component that supports complex layouts, branding, and status indicators using a flexible composition pattern.

## Overview

The `Sidebar` component provides the core scaffolding for a dashboard navigation system. By using specialized sub-components, you can easily build highly structured and interactive menus with minimal code.

<SidebarDemo />

## Import

```tsx
import { 
  Sidebar, 
  SidebarHeader, 
  SidebarFooter, 
  SidebarSection, 
  SidebarSectionItem 
} from '@/components/Sidebar/Sidebar'
```

## Installation

Add the sidebar into your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add sidebar
```
