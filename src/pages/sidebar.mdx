# Sidebar

A versatile and premium navigation sidebar component that supports complex layouts, branding, and status indicators using a flexible composition pattern.

## Import

```tsx
import { 
  SidebarHeader, 
  SidebarHeaderOpen,
  SidebarHeaderClose,
  SidebarFooter, 
  SidebarFooterOpen,
  SidebarFooterClose,
  SidebarSection, 
  SidebarItem 
} from '@/components/Sidebar/Sidebar'
```

## Installation

Add the sidebar into your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add sidebar
```

## Basic Usage

The `Sidebar` uses a semantic composition of headers, sections, and footers to create a cohesive navigation experience.

<InteractiveSidebar />

```tsx
import { Sidebar, SidebarHeader, SidebarSection, SidebarItem } from '@/components/dashkit';
import { FiLayout, FiActivity, FiSettings } from 'react-icons/fi';

export default function App() {
  const [activeTab, setActiveTab] = useState('dashboard');

  return (
    <Sidebar className="w-64 border-r">
      <SidebarHeader>
        <Logo />
        <span>Dashkit</span>
      </SidebarHeader>

      <SidebarSection title="General">
        <SidebarItem 
          icon={<FiLayout />} 
          active={activeTab === 'dashboard'}
          onClick={() => setActiveTab('dashboard')}
        >
          Overview
        </SidebarItem>
      </SidebarSection>
    </Sidebar>
  );
}
```

## Collapsible Sidebar

Enable the `collapsible` prop to show a floating toggle button on the sidebar edge. You can use the `isOpen` state to conditionally render content using native utility components.

```tsx
<Sidebar collapsible defaultOpen={true}>
  <SidebarHeader>
    <SidebarHeaderOpen>Full Branding</SidebarHeaderOpen>
    <SidebarHeaderClose>Logo</SidebarHeaderClose>
  </SidebarHeader>
  {/* ... items ... */}
</Sidebar>
```
