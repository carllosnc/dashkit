# Input

A primitive input component with support for labels, icons, error states, and helper text.

## Import

```tsx
import { Input } from '@/components/Input/Input'
```

## CLI

Add the input component to your project using the Dashkit CLI:

```bash
bunx dashkit-ui add input
```

## Basic Usage

The basic usage of the input with a label and placeholder.

<Preview>
  <Input label="Email Address" placeholder="Enter your email" />
</Preview>

```tsx
<Input 
  label="Email Address" 
  placeholder="Enter your email" 
/>
```

## With Icons

Use `leftIcon` and `rightIcon` to add icons to the input. The icons automatically transition their color when the input is focused.

<Preview>

      label="Search" 
      placeholder="Search documentation..." 
      leftIcon={<FiSearch />} 
    />
    <Input 
      label="Password" 
      type="password" 
      placeholder="••••••••" 
      rightIcon={<FiLock />} 
    />
  </div>
</Preview>

```tsx
import { FiSearch, FiLock } from 'react-icons/fi'

<Input 
  label="Search" 
  placeholder="Search..." 
  leftIcon={<FiSearch />} 
/>

<Input 
  label="Password" 
  type="password"
```

## Masking (Regex)

Use the `mask` prop to filter input values in real-time. Matches to the provided regex are replaced with an empty string.

<Preview>
  <div className="w-full max-w-sm">
    <Input 
      label="Only Digits" 
      placeholder="Try typing letters..." 
      mask={/[^\d]/g} 
    />
  </div>
</Preview>

```tsx
<Input 
  label="Only Digits" 
  mask={/[^\d]/g} 
  placeholder="123..." 
/>
```

## Common Patterns

You can use the `mask` and `formatter` props together to implement complex patterns like phone numbers and Brasil CPF.

<Preview>
  <InputDemo />
</Preview>

```tsx
const formatCPF = (value: string) => {
  return value
    .replace(/^(\d{3})(\d)/, '$1.$2')
    .replace(/^(\d{3})\.(\d{3})(\d)/, '$1.$2.$3')
    .replace(/\.(\d{3})(\d)/, '.$1-$2')
    .slice(0, 14);
};

<Input 
  label="CPF (Brasil)" 
  mask={/[^\d]/g}
  formatter={formatCPF}
  placeholder="000.000.000-00" 
/>
```

## Props

The component extends all native `InputHTMLAttributes<HTMLInputElement>` props, plus the following:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Input label rendered above the field. |
| `leftIcon` | `ReactNode` | — | Icon element rendered at the start of the field. |
| `rightIcon` | `ReactNode` | — | Icon element rendered at the end of the field. |
| `mask` | `RegExp` | — | Regex used to filter input values (matches are removed). |
| `formatter` | `(val) => string` | — | Function to format the value after masking. |
| `error` | `string` | — | Error message displayed below the field. |
| `helperText` | `string` | — | Helper message displayed below the field. |
| `className` | `string` | — | Additional CSS classes for the input element. |
| `ref` | `Ref<HTMLInputElement>` | — | Forwarded ref to the underlying `<input>` element. |
