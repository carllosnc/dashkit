# Modal

A centered, layered dialog component that focuses user attention on critical information or actions.

## Import

```tsx
import { Modal } from '@/components/Modal/Modal'
```

## Installation

Add the modal to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add modal
```

## Basic Usage

The modal is a controlled component toggled by the `isOpen` prop. It includes built-in support for titles, descriptions, and custom footers.

<Preview>
  <div className="flex flex-wrap gap-4">
    <BasicModal />
  </div>
</Preview>

```tsx
import { useState } from 'react'
import { Modal, Button } from '@/components/dashkit'

export default function App() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>Open Modal</Button>
      <Modal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        title="Project Overview"
        description="Review your latest project stats."
      >
        <p>Main content area...</p>
      </Modal>
    </>
  )
}
```

## Formats & Examples

Modals can be customized for different scenarios using sizes, alignments, and custom children.

### Alert Modal

Smaller modals are perfect for destructive actions or critical confirmations.

## Features

- **Spring-Based Animations**: Uses `framer-motion` for a premium scale/fade entry.
- **Backdrop Blur**: Softly blurs background content for better focus.
- **Body Scroll Lock**: Automatically prevents scrolling when the modal is active.
- **Portalled Rendering**: Renders at the document root to avoid layering issues.
- **Keyboard Accessible**: Closes instantly when the `Escape` key is pressed.

## Sizes

Predefined sizes to accommodate different content types: `sm`, `md` (default), `lg`, `xl`, `2xl`, and `full`.
