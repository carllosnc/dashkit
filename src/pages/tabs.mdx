# Tabs

A performant and premium tabs component with a smooth, fluid indicator animation powered by Framer Motion.

## Import

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from '@/components/Tabs/Tabs'
```

## CLI

Add the tabs to your project using the shadcn CLI:

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/carllosnc/dashkit/master/registry.json:tabs
```

## Animated Indicators

The indicator uses Framer Motion's `layoutId` to achieve a "sliding" effect when transitioning between tabs. This happens automatically when triggers are within the same `Tabs` root.

## Props

### Tabs

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `string` | — | The value of the tab that should be active on mount. |
| `value` | `string` | — | The controlled value of the active tab. |
| `onValueChange` | `(val: string) => void` | — | Callback fired when the active tab changes. |
| `children` | `ReactNode` | — | Content of the tabs container. |
| `className` | `string` | — | Additional CSS classes for the container. |

### TabsTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | A unique value that identifies this tab. |
| `children` | `ReactNode` | — | Content for the tab trigger button. |
| `className` | `string` | — | Additional CSS classes for the trigger button. |
