# Tabs

A performant and premium tabs component with a smooth, fluid indicator animation powered by Framer Motion.

## Import

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from '@/components/Tabs/Tabs'
```

## Installation

Add the tabs to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add tabs
```

## Basic Usage

The component is highly accessible and supports a fluid layout for its triggers.

<div className="not-prose">
  <Tabs defaultValue="account">
    <TabsList className="w-full mb-8">
      <TabsTrigger value="account">Account</TabsTrigger>
      <TabsTrigger value="password">Password</TabsTrigger>
    </TabsList>
    <TabsContent value="account">
      <h3 className="text-lg font-bold text-ds-900 dark:text-white mb-2">Account Settings</h3>
      <p className="text-sm text-muted-foreground">Update your account email and personal information.</p>
    </TabsContent>
    <TabsContent value="password">
      <h3 className="text-lg font-bold text-ds-900 dark:text-white mb-2">Password</h3>
      <p className="text-sm text-muted-foreground">Change your password and manage your security settings.</p>
    </TabsContent>
  </Tabs>
</div>

```tsx
<Tabs defaultValue="account">
  <TabsList>
    <TabsTrigger value="account">Account</TabsTrigger>
```

## Animated Indicators

The indicator uses Framer Motion's `layoutId` to achieve a "sliding" effect when transitioning between tabs. This happens automatically when triggers are within the same `Tabs` root.
