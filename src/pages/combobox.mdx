# Combobox

A combobox is a flexible select component that allows users to filter a large list of options via a search input.

## Import

```tsx
import { Combobox } from '@/components/Combobox/Combobox'
```

## Installation

Add the combobox component to your project using the Dashkit CLI:

```bash
bunx carllosnc/dashkit add combobox
```

## Interactive Guide

The combobox provides a powerful way to handle long lists of data. It filters the items as you type, and maintains accessibility by using the appropriate `listbox` and `input` roles.

## Basic Usage

The component behaves like a standard select, but with a search input.

<Preview>
  <div className="max-w-md w-full">
    <Combobox 
      label="Country Selection"
      placeholder="Select your country..."
      options={[
        { value: 'us', label: 'United States' },
        { value: 'ca', label: 'Canada' },
        { value: 'gb', label: 'United Kingdom' },
        { value: 'de', label: 'Germany' },
        { value: 'fr', label: 'France' },
      ]}
    />
  </div>
</Preview>

```tsx
<Combobox 
  label="Country Selection"
  placeholder="Select your country..."
  options={[
    { value: 'us', label: 'United States' },
    { value: 'ca', label: 'Canada' },
    { value: 'gb', label: 'United Kingdom' },
  ]}
/>
```


## Props

The combobox supports the following props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | An optional label displayed above the input. |
| `description` | `string` | — | Helper text displayed below the input. |
| `placeholder` | `string` | `'Search or select...'` | Text displayed when no value is selected. |
| `options` | `ComboboxOption[]` | — | An array of `{ value, label }` objects. |
| `value` | `string \| string[]` | — | The current selected value(s). |
| `onChange` | `(value: string \| string[]) => void` | — | Callback fired when option(s) are selected. |
| `disabled` | `boolean` | `false` | Whether the combobox is interactive. |
| `className` | `string` | — | Additional CSS classes for the container. |
| `multiple` | `boolean` | `false` | If `true`, allows selecting multiple options using Chips. |
| `clearSearchOnSelect` | `boolean` | `false` | If `true`, the search query will be cleared when an item is selected. |
