# Color Picker

A premium color picker component that allows users to select from a curated list of presets or provide a custom hex value.

## Import

```tsx
import { ColorPicker } from '@/dashkit';
```

## Basic Usage

The Color Picker provides a clean popover interface with high-fidelity animations and state management.

<Preview>
  <ColorPicker 
    label="Theme Color" 
    defaultValue="#3B82F6" 
    helperText="Choose your brand color"
  />
</Preview>

```tsx
function App() {
  const [color, setColor] = useState('#3B82F6');
  
  return (
    <ColorPicker 
      label="Brand Color" 
      value={color} 
      onChange={setColor} 
    />
  );
}
```
