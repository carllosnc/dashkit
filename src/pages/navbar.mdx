# Navbar

A premium, responsive navigation header with support for branding, navigation, and action buttons using a flexible composition pattern.

## Overview

The `Navbar` component provides a standard top-level container for application branding and navigation. By using several sub-components, you can easily compose complex layouts with full control over alignment and behavior.

## Installation

```bash
bunx carllosnc/dashkit add navbar
```

## Usage

```tsx
import { Navbar, NavbarBrand, NavbarLinks, NavbarActions, Button } from '@/components/dashkit';
import { FiSearch, FiBell } from 'react-icons/fi';

export default function Header() {
  return (
    <Navbar>
      <NavbarBrand>
        <LogoIcon className="h-6" />
        <span className="font-bold">MyDashboard</span>
      </NavbarBrand>

      <NavbarLinks>
        <a href="/overview" className="text-sm font-medium">Overview</a>
        <a href="/settings" className="text-sm font-medium text-base-500 hover:text-base-900 transition-colors">Settings</a>
      </NavbarLinks>

      <NavbarActions>
        <button className="text-base-400 hover:text-base-900 transition-colors">
          <FiSearch size={18} />
        </button>
        <Button size="sm">Get Started</Button>
      </NavbarActions>
    </Navbar>
  );
}
```

## Components

The Navbar library includes several specialized sections:

| Component | Description |
| :--- | :--- |
| `Navbar` | The root container that handles height, backdrop blur, and sticky behavior. |
| `NavbarBrand` | Used for logos, brand names, and mobile menu triggers. |
| `NavbarLinks` | A responsive container for navigation items (hidden on mobile). |
| `NavbarActions` | Container for utility icons, themes, and primary action buttons. |

## Props

The root `Navbar` component accepts the following styling props:

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `sticky` | `boolean` | `true` | Whether the navbar sticks to the top of the viewport. |
| `blur` | `boolean` | `true` | Whether to apply glassmorphism backdrop blur. |
| `className` | `string` | `-` | Custom classes for the header container. |
