# Navbar

A premium, responsive navigation header with support for branding, navigation, and action buttons using a flexible composition pattern.

## Overview

The `Navbar` component provides a standard top-level container for application branding and navigation. By using several sub-components, you can easily compose complex layouts with full control over alignment and behavior.

<Preview>
  <NavbarDemo />
</Preview>

## Installation

```bash
bunx carllosnc/dashkit add navbar
```

## Usage

```tsx
import { Navbar, NavbarBrand, NavbarLinks, NavbarActions, Button } from '@/components/dashkit';
import { FiSearch, FiBell } from 'react-icons/fi';

export default function Header() {
  return (
    <Navbar>
      <NavbarBrand>
        <LogoIcon className="h-6" />
        <span className="font-bold">MyDashboard</span>
      </NavbarBrand>

      <NavbarLinks>
        <a href="/overview" className="text-sm font-medium">Overview</a>
        <a href="/settings" className="text-sm font-medium text-neutral-500 hover:text-neutral-900 transition-colors">Settings</a>
      </NavbarLinks>

      <NavbarActions>
        <button className="text-neutral-400 hover:text-neutral-900 transition-colors">
          <FiSearch size={18} />
        </button>
        <Button size="sm">Get Started</Button>
      </NavbarActions>
    </Navbar>
  );
}
```
